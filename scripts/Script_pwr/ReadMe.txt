Prerequisites:
**************
1.Java 

Procedure:
***********
1.Copy all the .csv files to Script_Pwr folder.
2.Enter the names of the .csv files in Scenario.txt file.

Note:
Make sure that name of the CSV file should be without any spaces. 
There is no space at end of the csv file name in scenarios.txt file

3.Run the perl script Pwr_values.pl IW/MSDK/ADV

Command Line Argument for Inwatch is IW.
Command Line Argument for MSDK1/2/3 is MSDK
Command Line Argument for Advanced is ADV.
