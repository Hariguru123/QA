1. mail_on_off.pl
******************

Description:  This script is used to turn ON/OFF mail notifications while building the dhanush targets. This will generate mail.txt file with some content based on the user input.

Usage: perl mail_on_off.pl

Ex: If user input is 1, data in the text file will be "Buildfail ON  Buildpass ON", build script read this line to turn ON/OFF mail notifications.
